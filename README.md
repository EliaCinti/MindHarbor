# MindHarbor
MindHarbor  MindHarbor è un’applicazione che facilita la gestione delle relazioni tra psicologi e pazienti. I pazienti possono prenotare appuntamenti, visualizzare la lista dei propri appuntamenti e completare test prescritti. Gli psicologi possono gestire appuntamenti, visualizzare le schede personali dei pazienti e prescrivere test e terapie.
