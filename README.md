# MindHarbor
MindHarbor è un'applicazione che facilita la gestione delle relazioni tra psicologi e pazienti. I pazienti possono prenotare appuntamenti, visualizzare la lista dei propri appuntamenti e completare test prescritti. Gli psicologi possono gestire appuntamenti, visualizzare le schede personali dei pazienti e prescrivere test e terapie.

## Funzionalità

### Per i Pazienti
- Prenotazione appuntamenti
- Visualizzazione degli appuntamenti passati e futuri
- Esecuzione di test prescritti

### Per gli Psicologi
- Gestione degli appuntamenti
- Visualizzazione delle schede personali dei pazienti
- Prescrizione di test e terapie

## Tecnologie Utilizzate
- **Frontend**: JavaFX per l'interfaccia grafica
- **Backend**: Java per la logica di business
- **CLI**: Interfaccia a riga di comando
- **Persistenza**: Supporto a doppia persistenza con MySQL e CSV
- **Testing**: Test con JUnit
